About This Dashboard

Understanding federal workforce data through interactive visualization

Overview

This interactive dashboard provides comprehensive analysis of the U.S. federal civilian workforce. It visualizes employment data across agencies, geographic locations, pay grades, and demographic categories to help researchers, policymakers, and the public understand the composition and distribution of federal employees.

Data Source

All data in this dashboard comes from the U.S. Office of Personnel Management (OPM) FedScope database. FedScope is OPM's official source for federal workforce statistics and provides public access to data about federal civilian employees.

Note: Some salary data appears as "REDACTED" in the source data to protect employee privacy in cases where revealing the salary could identify specific individuals. This is standard OPM practice for positions with very few employees.

Data Snapshot: November 2025
Total Records: 2+ million employment records
Coverage: All federal civilian agencies

How to Use This Dashboard

Navigation Tabs

Overview

High-level summary charts showing top agencies, salary distributions, and workforce composition.

Agencies

Searchable table of all federal agencies with employee counts, salaries, tenure, and grade data.

Geography

State-by-state analysis with interactive charts, maps, and comparison tools.

Demographics

Age distribution, education levels, and salary breakdowns by demographic category.

Compare

Side-by-side comparison tool for analyzing differences between agencies.

Interactive Features

  • Global Filters: Use the filter bar at the top to narrow data by agency, state, or pay range. All charts and tables update automatically.
  • Click to Filter: Click on any bar in the charts or row in the tables to filter the entire dashboard to that selection.
  • Sortable Tables: Click column headers to sort tables by any metric.
  • Search: Use search boxes to quickly find specific agencies or states.
  • Chart/Map Toggle: In the Geography tab, switch between bar chart and map views.

Key Metrics Explained

  • Employee Count: Total number of federal civilian employees (not including military or contractors).
  • Average Salary: Mean annualized adjusted basic pay across all employees in the category.
  • Median Salary: The middle value when all salaries are arranged in order (less affected by outliers than average).
  • Average Tenure: Mean years of federal service for employees in the category.
  • Average Grade: Mean GS (General Schedule) grade level where applicable.
  • STEM Occupations: Positions classified by OPM as Science, Technology, Engineering, or Mathematics related.

Technical Notes

This dashboard was built using vanilla HTML, CSS, and JavaScript with Chart.js for visualizations. The underlying data was processed using Python (pandas) to aggregate the raw OPM data into optimized summary files for fast browser-based rendering.

The source code and Jupyter notebook used for data processing are available in the GitHub repository.
